Pengertian bahasa pemograman python adalah bahasa pemograman tinggi yang dapat melakukan eksekusi sejumlah instruksi multi guna secara langsung (interpretatif) serta menggunakan semantic dinamis untuk memberikan tungkat keterbacaan syntax. Sebagai bahasa pemograman tinggi, python dapat dipelajari dengan mudah karana sudah dilengkapi dengan manajemen memori otomatis (pointer).
Pada dasarnya, bahsa pemograman yang dirancangoleh Guido Van Rossum ini sebenarnya sangat banyak digunakan untuk membuat program yang sering dipakai dimasyarakat umum. Python dapat digunakan secara bebas, bahkan untuk kepentingan komersial sekalipun. Banyak perusahaan yang mengembangkan bahsa pemograman python secara komersial untuk memeberikan layanan.
